# 🏨 Hostel Management System

A full-stack web application to manage hostel bookings efficiently.

---

## 🚀 Features
- Add Booking
- View Bookings
- Delete Booking
- Automatic Cost Calculation

---

## 🛠 Tech Stack
- HTML, CSS
- PHP
- MySQL

---

## ▶️ Run Locally
1. Install XAMPP
2. Move project to `htdocs`
3. Start Apache & MySQL
4. Open:
   http://localhost/INT220/hostel-management/
